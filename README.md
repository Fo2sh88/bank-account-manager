# Bank Account Manager

A simple Python OOP project to manage bank accounts.

## Features
- Create accounts with a name and initial balance
- Deposit money
- Withdraw money (with balance checks)
- Check balance
- View transaction history

## Project Structure
- `account.py` — Account class definition
- `main.py` — Main program and user interface
- `test_account.py` — Unit tests

## How to Run
```bash
python main.py
```

## How to Test
```bash
python -m pytest test_account.py
# or
python -m unittest test_account.py
```
